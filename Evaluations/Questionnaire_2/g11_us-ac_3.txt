User Story: 
As a VisualDesigner, I want to create a moodboard to aid with visual design concepts for the nsf.gov website, so that we can discuss different design concepts with stakeholders.

Acceptance Criteria (generated from Model #1)
1. The moodboard creation feature should be accessible from the VisualDesigner's account dashboard.
2. The VisualDesigner should be able to add, remove, and rearrange images within the moodboard.
3. The VisualDesigner should be able to upload images from their device or select images from a pre-existing library.
4. The moodboard should support various file formats, including JPEG, PNG, and GIF.
5. The VisualDesigner should be able to add notes or descriptions to each image on the moodboard.
6. The moodboard should have a zoom functionality to allow the VisualDesigner to view the images in detail.
7. The VisualDesigner should be able to share the moodboard with stakeholders via a unique URL or by sending email invitations.
8. The stakeholders should be able to view the moodboard without needing to create an account or log in.
9. The moodboard should display images and their accompanying notes in a visually appealing and organized manner.
