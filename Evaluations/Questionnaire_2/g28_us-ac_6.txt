User Story: 
As a Zooniverse admin, I want to know when I should educate a volunteer.	

Acceptance Criteria (generated from Model #1)
1. The system should track and identify volunteers who consistently provide incorrect or incomplete answers.
2. When a volunteer's accuracy falls below a predefined threshold, the system should generate an alert for the admin.
3. The admin should receive a notification or email indicating which volunteer needs to be educated.
4. The notification or email should include specific information regarding the volunteer's performance and areas where they require education.
5. The system should keep a record of all notifications sent to the admin for tracking purposes.
6. The admin should have the ability to mark a notification as resolved or indicate that the volunteer has been educated.
7. If a previously educated volunteer's performance drops again, a new notification should be generated for the admin.

Acceptance Criteria (generated from Model #2)
1. The Zooniverse admin should receive a notification when a volunteer consistently provides incorrect or inconsistent answers.
2. The notification should include details such as the volunteer's username, the project they are working on, and the specific tasks or questions they are struggling with.
3. The admin should be able to view a summary or report of all volunteers who require education.
4. The system should track and analyze the volunteer's performance over time to determine when education is needed.
5. The admin should have the ability to manually trigger an education notification for a specific volunteer if necessary.
6. The notification system should support different communication channels, such as email or in-app notifications, to ensure the admin receives timely alerts.
7. The system should provide a clear and intuitive interface for the admin to review the volunteer's performance history and identify any patterns or trends indicating a need for education.

Acceptance Criteria (generated from Model #3)
1. The Zooniverse admin should receive notifications when a volunteer consistently provides incorrect or inconsistent answers.
2. The notification should include the specific task or project in which the volunteer is struggling.
3. The admin should be able to access detailed information about the volunteer's performance in order to identify areas for education and improvement.
4. The system should track the number of incorrect or inconsistent answers from each volunteer over a specified period of time.
5. The admin should have the ability to set thresholds for triggering the notification based on the number or percentage of incorrect or inconsistent answers.
6. The system should allow the admin to record and save educational content or resources for each task or project.
7. The admin should be able to assign and track the completion of educational modules or quizzes by volunteers.
8. The system should provide a summary report of each volunteer's educational progress and improvements made over time.
