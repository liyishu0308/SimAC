User Story: 
As a depositor, I want to allow my collaborators privileged access to datasets, so that we continue to have a productive relationship.

Acceptance Criteria:
1. The depositor should be able to grant and manage privileged access to datasets for the collaborators.
2. The depositor should be able to manage the level of privileges to the collaborators on specific datasets (read-only, edit, or adminstrator).
3. The collaborators should receive notifications when the privileges assigned to them are updated.
4. The collaborator should be able to apply for the previledge on specific dataset. The application will be notified to depositor.
5. The depositor should be able to approve or reject the application.
6. The previledge should be active immediately once the depositor granted access right or approved the application.
7. The depositor should be able to modify the priviliges.
8. The depositor should be able to define dataset access at the file or folder level.
