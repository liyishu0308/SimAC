User Story: 
As an author, I want to be able to indent / outdent content.

Acceptance Criteria:
1. The text editor should have button of "Indent" or "Outdent".
2. The text editor should have keyboard shortcuts for "Indent" or "Outdent".
3. The user should be able to select texts (from multiple pargraph or lines) to indent/outdent them.
4. The user should be able to customize the indenting/outdenting levels in specific margin.
5. The user should be able to define the keyboard shortcuts for "Indent"/"Outdent".
6. After indenting/outdenting, the contents will be updated in real-time.
7. The format of the content will not change after indenting/outdenting.
8. The system should have option for restoring the initial format.
